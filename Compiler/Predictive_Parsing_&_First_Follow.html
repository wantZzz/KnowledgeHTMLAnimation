Chapter 4.4 Top-Down Parsing

分析環境與集合計算

待分析輸入字串： b
文法規則 (CFG)：
S → A B
A → a | ε
B → b | c
預先計算之集合：
FIRST(S) = { a, b, c }
FIRST(A) = { a, ε }
FIRST(B) = { b, c }
FOLLOW(A) = FIRST(B) = { b, c }
